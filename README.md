The Movie Recommendation System is a web based application that helps users make movie selections according to personal criteria such as genre preference, mood preference, availability of time, and preference towards popularity.
This project is built using Flask for backend, and HTML/CSS/JavaScript for frontend.
This project does not use machine learning algorithms for recommendation. Instead, it uses rule based recommendation, making the process lightweight and efficient.
The project follows the Agile Software Development Model with iterative improvements.
The Agile model was chosen because:
Requirements evolved during UI and feature design.
Incremental development allowed testing each module.
Backend and frontend integration required continuous iteration.
Flexibility was needed for API integration and UI refinement for future improvements.
